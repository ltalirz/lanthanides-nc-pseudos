CP2K input template:
#+name: template
#+begin_example
&GLOBAL
  PROGRAM_NAME ATOM
&END GLOBAL
&ATOM
  ELEMENT  {0:s}
  ELECTRON_CONFIGURATION {1:s}
  CORE [Xe]
  &METHOD
     METHOD_TYPE  KOHN-SHAM
     &XC
       &XC_FUNCTIONAL PBE
       &END XC_FUNCTIONAL
     &END XC
  &END METHOD
  &PP_BASIS
    BASIS_TYPE GEOMETRICAL_GTO
  &END PP_BASIS
  &POTENTIAL
    PSEUDO_TYPE GTH
    &GTH_POTENTIAL
    {2:s}
    &END
  &END POTENTIAL
  &PRINT
    &ANALYZE_BASIS
         OVERLAP_CONDITION_NUMBER T
         COMPLETENESS T
    &END ANALYZE_BASIS
    &UPF_FILE
      FILENAME {0:s}.UPF
    &END
  &END
&END ATOM

#+end_example

Read GTH data from each directory (atom) and write cp2k ATOM input to =[Atom].inp=:
#+begin_src python :var data=electron_configurations template=template :results output
  import pandas as pa
  import os

  df = pa.DataFrame([{'type': x[0], 'config': x[1]} for x in data[1:]])
  # open  cp2k GTH data
  rows = []
  for _, row in df.iterrows():
      atom = row['type']
      row['gth'] = '    '.join(open(os.path.join(atom,atom), 'r').readlines()[1:])
      rows.append(row)
  df = pa.DataFrame(rows)
  for _, row in df.iterrows():
      out = template.format(row['type'], row['config'], row['gth'])

      with open(os.path.join(row['type'], row['type']+'.inp'), 'w') as f:
          f.write(out)
